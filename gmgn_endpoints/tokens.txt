# Tokens Endpoints

## Endpoints Related to Tokens
| **Endpoint**                              | **Purpose / Possible Use**                                      |
|-------------------------------------------|-----------------------------------------------------------------|
| `/tokens/sol`                             | General information about tokens on Solana                      |
| `/tokens/top_traders/sol`                 | Top traders of a specific Solana token                          |
| `/tokens/top_buyers/sol`                  | Top buyers of a specific Solana token                           |
| `/tokens/security/sol`                    | Security details of a specific token                            |
| `/tokens/sol/create`                      | Endpoint to create a new Solana token                           |
| `/tokens/sol/update`                      | Endpoint to update a specific Solana token                      |
| `/tokens/sol/delete`                      | Endpoint to delete a specific Solana token                      |
| `/tokens/sol/details`                     | Detailed information about a specific token                     |
| `/tokens/sol/metadata`                    | Metadata information for a specific Solana token                |
| `/tokens/sol/metrics`                     | Metrics and analytics for a specific Solana token               |
| `/tokens/sol/prices`                      | Pricing data for a specific Solana token                        |
| `/tokens/sol/historical`                  | Historical price data for a specific Solana token               |
| `/tokens/sol/analytics`                   | Analytics and insights for a specific Solana token              |
| `/tokens/sol/alerts`                      | Alerts related to a specific Solana token                       |
| `/tokens/sol/info`                        | General info about a specific Solana token                      |
| `/tokens/sol/stats`                       | Statistics and aggregated data about a Solana token             |
| `/tokens/sol/history`                     | History of changes or transactions for a Solana token           |
| `/tokens/security/sol/audit`              | Security audit details for a Solana token                       |
| `/tokens/security/sol/status`             | Current security status of a Solana token                       |
| `/tokens/security/sol/report`             | Security reports related to a Solana token                      |
| `/tokens/security/sol/alerts`             | Alerts related to security of a specific token                  |
| `/tokens/top_traders/sol/details`         | Details about top traders of a specific Solana token            |
| `/tokens/top_traders/sol/history`         | Historical trading data of top traders                          |
| `/tokens/top_buyers/sol/history`          | Historical data of top buyers for a Solana token                |
| `/tokens/top_buyers/sol/details`          | Detailed info on top buyers of a specific Solana token          |
| **New** `/tokens/monthly`                 | Monthly summary and data for a specific token                   |
| **New** `/tokens/security/sol/status`     | Security status of tokens, updated monthly                      |
| **New** `/tokens/security/sol/report`     | Comprehensive monthly security report for a token               |

| `/smartmoney/delete`                      | Delete a SmartMoney entity                                      |
| `/smartmoney/update`                      | Update SmartMoney details                                       |

| `/smartmoney/stats`                       | General statistics for SmartMoney                               |
| `/smartmoney/alerts`                      | Alerts related to SmartMoney                                    |
| `/smartmoney/realtime`                    | Real-time SmartMoney data                                       |
| `/smartmoney/report`                      | Generate a SmartMoney report                                    |
| `/smartmoney/analytics`                   | Analytics for SmartMoney activities                             |
| `/smartmoney/summary`                     | Summary information for SmartMoney                              |
| `/smartmoney/monthly`                     | Monthly data related to SmartMoney                              |
| `/smartmoney/yearly`                      | Yearly data related to SmartMoney                               |
| `/smartmoney/metrics`                     | SmartMoney performance metrics                                  |
| `/smartmoney/pricing`                     | Pricing information for SmartMoney entities                     |
| `/smartmoney/details`                     | Get detailed SmartMoney data                                    |

# Trades Endpoints

## Endpoints Related to Trades
| **Endpoint**                              | **Purpose / Possible Use**                                      |
|-------------------------------------------|-----------------------------------------------------------------|
| `/trades/sol`                             | General trade data for a specific Solana token                  |
| `/trades/sol?limit=100`                   | Trade data with a limit parameter                               |
| `/trades/sol?tag[]=creator&tag[]=dev_team`| Trades tagged as creator or dev team transactions               |
| `/trades/sol/create`                      | Endpoint to create a new trade                                  |
| `/trades/sol/update`                      | Endpoint to update an existing trade                            |
| `/trades/sol/delete`                      | Endpoint to delete a trade                                      |
| `/trades/sol/metrics`                     | Metrics and analytics for specific trades                       |
| `/trades/sol/history`                     | Historical trade data                                           |
| `/trades/sol/analytics`                   | Analytics and insights about trades                             |
| `/trades/sol/stats`                       | General statistics about trades                                 |
| **New** `/trades/sol/monthly`             | Monthly trade statistics                                        |
